Overview

The Wunderbar Multi-Programmer tool is designed to program up-to 3 devices using the Atmel-ICE programmer. The tool is configured through a set of JSON configuration files.

Configuration of the tool for programming the RSD 156 Auto Saucer has been provided.

Work Flow

Normal operation of this tool involves the following steps:

  1. Load the configuration from the .json file.
  2. Connect each of the programmers via JTAG to the device to be programmed.
  3. Hit the Program button to begin the programming.
  4. Wait for the programmers to complete programming and verification.

The last settings will be remembered by the programming tool.
