RSD 156 Connection Reference

The RSD Pizza Saucer can be programmed using the Wunderbar Multi-Programmer tool. This help file describes how to connect the Atmel-ICE JTAG programmers to the RSD device.

Atmel-ICE JTAG Programmer

Connect each of the 3 Atmel-ICE devices to the USB hub and plug the USB hub into the host system running the Wunderbar Multi-Programmer. Each Atmel-ICE device has a label identifying the serial # of the device. This serial # is used by the programming software to program a particular firmware to each of the three devices.

Atmel-ICE JTAG Programmer
