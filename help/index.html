Wunderbar Multi-Programmer Help

This is the documentation for the Wunderbar Multi-Programmer tool which is designed to program up-to 3 devices using the Atmel-ICE programmer.



Work Flow

Normal operation of this tool involves the following steps:

  1. Load the configuration from the .json file.
  2. Connect each of the programmers via JTAG to the device to be programmed.
  3. Hit the Program button to begin the programming.
  4. Wait for the programmers to complete programming and verification.
